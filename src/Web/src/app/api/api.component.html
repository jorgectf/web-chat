API

Web API written in .NET Core 3.1 (C# 8).
Microsoft SQL Server 2019 is used as a DBMS.

SPA application is written on Angular using Bootstrap (adapted for Angular package - ng-bootstrap).

Detailed specification Swagger, you can find here.

Overview

General information about important things.

Protocol

To keep the connection of clients with the server, the SignalR library is used, which implements the work of clients over the WebSocket protocol. The SqlTableDependency library is used to notify clients about changes made not through the server, but through the DBMS, for example.

Architecture

On the server is implemented CQRS architecture.

Authentication

Authentication is implemented through JWT tokens. Verification and issuance of tokens takes place on the same server, without using IdentityServer.
